# IsoCampus_Erebus

> Building my college campus from scratch using Python and Pygame.

---

## About

I have always had a strange dislike for Unity and Unreal Engine. I don't really know why. Maybe it's because everything is already built for you, while I have always enjoyed writing things from scratch in Pygame.

Most of my projects have been 2D pixel games. Recently, I started experimenting with isometric rendering and simple 3D concepts in Pygame. That's when I thought,

**"Why not recreate my college campus?"**

So, this project was born.

**IsoCampus_Erebus** is my attempt to build an isometric version of my college campus entirely in Python and Pygame while learning graphics programming and rendering techniques along the way.

I'm also putting this project on GitHub so I stay consistent and actually finish it.
